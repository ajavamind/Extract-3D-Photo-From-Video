H - Show/Remove Keyboard Help Command List
C - Change color of Text and Crosshairs

I - Select input video filename
O - Select output photo folder

Y - Single Mode. Save single frame without a suffix
D - 3D Mode. Save _L and _R frames only
V - 4V Mode. Save _LL _LM _RM _RR frames for LumePad 3D Tablet
T - Lenticular Mode. Save frames with suffix _01 to _10 (Not implemented)

Space     - Next Video Frame
Backspace - Previous Video Frame

L         - Set Frame Type as _L (3D mode) or _LL (4V mode)
M         - Set Frame Type as _LM (4V mode)
N         - Set Frame Type as _RM (4V mode)
R         - Set Frame Type as _R (3D mode) or _RR (4V mode)
0-9       - Set Frame Type as Lenticular _00 to _09 (Lenticular mode)

S         - Save Current Display Frame with filename defined by Frame Type
          - Save Frame with suffix _L (3D mode) or _LL (4V mode)
          - Save Frame with suffix _LM (4V mode)
          - Save Frame with suffix _RM (4V mode)
          - Save Frame with suffix _R (3D mode) or _RR (4V mode)
          - Save Frame with suffix _00 to _09 (Lenticular mode)
		  
+         - Advance Group filename counter
-         - Reduce Group filename counter

J         - JPG output 
P         - PNG output (default)

Mouse click sets crosshairs location at cursor 
   (do only once per 3d or 4V frame group)
   (crosshairs set the stereo window for anaglyph)
   
For current video frame only:
   Left Arrow  - shift display left 1 pixel
   Right Arrow - Shift display right 1 pixel
   Up Arrow    - Shift display up 1 pixel
   Down Arrow  - Shift display down 1 pixel

A         - Toggle Display current saved 3D frames (_L and _R) as Anaglyph
X         - Swap Left and Right Images in 3D Anaglyph Display Mode
[         - Reduce crosshair spacing percent of screen width
]         - Increase crosshair spacing percent of screen width

Enter/Play/Pause  Play frames at 5 frames per second
